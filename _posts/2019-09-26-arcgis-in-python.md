## Working with imagery
set item type to: ```Imagery Layer```
```python
import arcgis
from arcgis.gis import GIS
from IPython.display import display

gis = GIS()
```
```
items = gis.content.search("Landsat 8 Views", item_type="Imagery Layer", max_items=2)
```
